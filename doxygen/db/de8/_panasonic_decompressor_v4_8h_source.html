50  // The RW2 raw image buffer is split into sections of BufSize bytes.
51  // If section_split_offset is 0, then the last section is not necessarily
52  // full. If section_split_offset is not 0, then each section has two parts:
53  // bytes: [0..section_split_offset-1][section_split_offset..BufSize-1]
54  // pixels: [a..b][0..a-1]
55  // I.e. these two parts need to be swapped around.
